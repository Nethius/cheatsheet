* **G++ флаги**: <a name="1"></a>
    * **-L** (большая л) - указание каталога для поиска библиотек.
    * **-l** (маленькая л) - указание конкретной библиотеки для линковки
    * **-I** (большая и) - указание каталога для поиска подключаемых файлов.
